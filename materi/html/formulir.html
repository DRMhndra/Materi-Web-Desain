Form (Formulir)

Form adalah element html yang digunakan untuk mengumpulkan informasi dari seorang pengguna.


 
 ......

form memiliki attribute wajib yaitu action dan method

  • action digunakan untuk mengirim hasil informasi ke tempat tertentu/server
  • method adalah cara pengiriman data bisa menggunakan POST ataupun GET


Formulir dengan metode POST


Perbedaan Method POST dan GET

GET
Method GET akan menampilkan informasi di dalam url bar
Method GET digunakan untuk perpindahan halaman dalam HTML
POST
Method POST tidak akan menampilkan informasi di url bar
Method POST digunakan untuk menyimpan informasi yang bersifat privasi

Didalam form dapat menggunakan berbagai macam element input untuk mengumpulkan informasi di antaranya:

  • text
  • radio
  • checkbox
  • password
  • tel
  • number
  • submit
  • dll
