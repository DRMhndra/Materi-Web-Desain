Link

Link adalah elemen html yang berguna untuk kita pindah ke halaman lain atau ke sebuah situs website.

 ini sebuah link
ini sebuah link

Attributehref berguna untuk mengarahkan link ke target tertentu,attibute optional yaitu target untuk membuka link di tab baru.


Terdapat 3 jenis link, diantaranya:

  • anchor untuk mengarahkan di halaman yang sama
  • relative url mengarahkan ke page yang berbeda namun masih di website yang sama
  • absolute url mengarahkan ke website yang berbeda
