Semantic Tag

Semantic tag adalah tag html5 yang digunakan untuk mengelompokan element-element tertentu dan untuk membuat layout di semantic tag terdapat 7 tag yang biasa digunakan:

  1. header
  2. nav
  3. main
  4. aside
  5. article
  6. section
  7. footer

Semantic tag dapat digunakan sesuai dengan kebutuhan, sebelum ada semantic tag,menggunakan div dan span untuk mengelompokan kode.
