Element HTML

HTML terdiri dari banyak elemen,sebuah elemen HTML ditulis menggunakan tag awal dan tag akhir, dan dengan konten di antaranya.

Sebuah element html dapat berisi element html yang lainnya atau bisa disebut nested element.

Self-closing Tag

Self-closing Tag atau yang disebut tag penutup diri hanya memiliki tag pembuka,ada beberapa element html yang bersifat self-closing tag diantaranya:

Element Self-closing tag biasanya memiliki attribute tambahan untuk memberikan informasi.

Attribute

Attribute adalah informasi tambahan yang terdapat dalam element html,bisanya setiap attribute memiliki sebuah value atau isi.

Beberapa element html yang harus memiliki attribute wajib

  1. a atau link,attribute wajib yaitu href
  2. img attribute wajib yaitu src dan alt
  3. input attribue wajib type dan name
  4. form attribute wajibnya action dan method

Komentar

Komentar berguna untuk memberi informasi tambahan soal kode yang kita tuliskan,mempermudah orang lain membaca serta mengerti ketika melihat kode tersebut

Komentar tidak akan di pernah ditampilkan oleh web browser ketika dijalankan tetapi akan tetap terlihat ketika melakukan inspect element,ctrl + / adalah short-cut membuat komentar.
