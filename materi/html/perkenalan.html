HTML

HTML adalah singkatan dari Hyper Text Markup Language.Tidak seperti bahasa pemrograman yang menggunakan script untuk menjalankan codenya,bahasa markup menggunakan tag untuk mengidentifikasi konten.

Struktur WEB

  • HTML : Struktur Dasar Website
  • CSS : Presentasi & mempercantik Website
  • JS : Perilaku dari suatu element

Strukur untuk membuat HTML

<!DOCTYPE html>
<html>
<head>
 <title></title>
</head>
<body>

</body>
</html>

Struktur dokumen HTML bertipe sandwich,karena bertipe layaknya sebuah sandwich maka memiliki pembuka dan penutup.

  • !DOCTYPE html : digunakan memberitahu web broswer bahwa dokumen yang akan di muat bertipe html
  • html : digunakan membuat sebuah dokumen html
  • head : berisi informasi tentang web
  • title : digunakan judul dari html
  • body : struktur utama dalam membuat html,element yang kita buat akan ditampilkan di web browser

File yang dibutuhkan untuk belajar web desain

  1. WEB Server (optional) contoh: Xampp,lamp,ataupun mamp

Cara membuat file HTML

Untuk membuat file html dapat menggunakan notepad tetapi untuk notepad belum tersedia auto-completion sehingga mengharuskan menuliskan pembuka serta penutup secara manual,agar file teridentifikasi sebagai HTML dapat menggunakan ekstensi .html

<!DOCTYPE html>
<html>
<head>
 <title></title>
</head>
<body>
 
Hello World!

</body>
</html>
