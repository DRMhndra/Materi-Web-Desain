Konsep yang sangat penting dalam javascript adalah fungsi dikarenakan hampir semua syntax di dalam javascript berupa fungsional.


Fungsi juga dapat digunakan untuk tugas tertentu sehingga tidak menuliskan perintah yang sama dalam suatu kondisi.

Syntax

//fungsi tanpa paramater
function nama_function() {
 // statment yang dijalankan
}
//fungi dengan paramater
function nama_function(< paramter1, paramater2, ... >) {
 // statment yang dijalankan
}

Setiap fungsi pasti memiliki nilai balik atau return yang dapat ditampung oleh variable lain.

Function Chain

Fungsi juga dapat dikaitkan satu sama lain agar mempersingkat suatu syntax sehingga tidak diperlukannya variable tambahan untuk menampung semua nilai dari fungsi.
