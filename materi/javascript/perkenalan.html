Javascript

Javascript adalah bahasa pemgrograman yang dapat digunakan untuk meningkatkan interaksi user dengan halaman web, penggunaan javascript dapat membuat halaman lebih interaktif serta dinamis. Javascript dapat berjalan di kedua sisi yaitu Back-end dan Front-end.

Sejarah

Javascript diciptakan oleh Brendon Eich pada tahun 1955, pengembangan javaScript hanya memakan waktu 10 hari pembuatan dengan nama awal yaitu LiveScript hingga kini menjadi javascript.


Dalam menjalankan code javascript tidak memerlukan compiler hanya butuh aplikasi web browser, javascript lebih mudah dalam dipelajari dan kini dapat dijalankan di berbagai macam platform.

Interpreted VS Compiled

Dalam pemrograman terdapat 2 cara menjalankan kode diantaranya:

Interpreted

Kode akan dieksekusi saat dijalankan tanpa harus dilakukan perubahan terlebih dahulu ke format tertentu contohnya adalah Javascript, Sql, Python.

Compiled

Kode akan di ubah terlebih dahulu ke bentuk atau format lain sebelum dijalankan oleh processor atau interpreter bahasa tersebut contohnya C++ dan Java.
