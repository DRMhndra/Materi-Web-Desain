Penambahan javascript dalam file dapat di lakukan dengan cara seperti css, yang membedakan untuk css penempatan menggunakan tag link sedangkan javascript menggunakan tag script.


Cara menambahkan file javascript

Internal

Penulisan kode javascript secara internal dituliskan didalam file yang sama di dalam tag script ditempatkan sebelum tag penutup body.

Lihat Kode

Eksternal

Penulisan kode javascript secara terpisah di dalam suatu file tersendiri lalu akan dihubungkan dengan file menggunakan tag script dengan tambahan attribute src untuk menentukan dimana lokasi file tersebut.

Lihat Kode

Inline

Penulisan kode secara inline sedikit berbeda dikarenakan menulisnya secara langsung sebagai attribute dalam element html.

Lihat Kode

Bisa dilihat didalam file html terdapat attribute event yang ada di javascript, ketika tombol ditekan akan menjalankan fungsi tampilPesan yang ada di dalam file javascript.
