let namaSaya = 'Dani Nur Rahman'; 
let NamaSaya = 'Ahmad Sobari';

Javascript bersifat Case Sensitif sehingga variable namaSaya dengan NamaSaya berbeda.

Variable

Variable atau yang biasa disebut tempat penyimpanan data sementara yang sewaktu-waktu isinya dapat diubah atau dimanipulasi sesuai dengan kondisi.


Hal yang pertama kali perlu diperhatikan dalam pembuatan variable adalah keyword, terdapat 3 keyword untuk mendeklarasikan variable dalam javascript diantaranya:

var

Keyword yang penggunaannya bersifat global dikarenakan tidak mengenal konsep blocking sehingga kini jarang digunakan dengan berbagai alasan.

let

Keyword yang telah mengenal sifat blocking dan let juga dapat melakukan penulisan ulang nlai dari sebuah variable.

const

Keyword yang tidak memungkingkan penulisan ulang nilai dari suatu variable.
