Operator berfungsi untuk melakukan suatu perintah atau intruksi yang diberikan seperti penugasan, aritmatika, dan logical.

Operator Aritmatika

Operator aritmatika digunakan untuk melakukan perhitungan sederhana.

Operator Penjelasan
+ Penjumlahan
- Pengurangan
* Perkalian
/ Pembagian
% Modulus
** Perpangkatan

Operator Penugasan

Operator penugasan tidak jauh berbeda dengan artimatika yang digunakan untuk melakukan operasi perhitungan sederhana dan memasukan nilai ke suatu variable.

Operator Contoh Penjelasan
= x = 1 Penugasan
+= x += y = x + y
-= x -= y = x - y
* x += y = x + y
/ x /= y = x / y
% x %= y = x % y
** x **= y = x ** y

Operator Perbandingan

Operator Penjelasan
== Sama Dengan
=== Sama Dengan (Tipe data)
!= Tidak sama dengan
!== Tidak sama dengan (Tipe data)
> Lebih dari
>= Lebih dari sama dengan
< Kurang dari
<= Kurang dari sama dengan

Operator Logical

Operator Penjelasan
&& And
|| Or
! Not
