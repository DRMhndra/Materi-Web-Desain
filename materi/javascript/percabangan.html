Percabangan adalah kondisi dimana program akan dijalankan berdasarkan kondisi tertentu yang telah di terapkan dan sesuai atau benar.


Terdapat 6 macam percabangan dalam javascript

  • If
  • If..Else
  • If..Elseif
  • If Nested
  • Switch
  • Ternary

If

Percabangan yang paling sederhana, kondisi if tunggal hanya akan djalankan sesuai jika kondisi benar.

if(< kondisi >){
 // statment yang akan dijalankan
}

If..Else

Percabangan yang satu ini akan menghandle apabila kondisi tidak terpenuhi sedikit berbeda dengan if tunggal.

if(< kondisi >){
 // kondisi bila benar
} else {
 // kondisi bila salah
}

If..elseif

Bentuk percabangan ini memiliki bentuk lebih kompleks, dimana terdapat kondisi didalamnya.

if(< kondisi >){
 // kondisi bila benar
} elseif(< kondisi >) {
 // kondisi lain
} elseif(< kondisi >) {
 // kondisi lain
} else {
 // kondisi bila semua kondisi tidak terpenuhi
}

Nested IF

Bentuk percabangan dimana terdapat percabangan lain di dalam suatu statment.

if(< kondisi >){
 // kondisi bila benar
} elseif(< kondisi >) {
  if(< kondisi >){
   // kondisi bila benar
  } else {
   // kondisi bila salah
  }
} elseif(< kondisi >) {
  if(< kondisi >){
   // statment yang akan dijalankan
  }

 // kondisi bila semua kondisi tidak terpenuhi

Ternary

Percabangan ternary adalah versi singkat dari percabangan if..else yang hanya satu baris.

(nilai yang dibandingkan) ? < "perintah bila benar" > : "perintah bila salah";
