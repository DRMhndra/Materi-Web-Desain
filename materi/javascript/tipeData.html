Didalam javascript tipe data dibedakan menjadi 2 yaitu Data Primitive dan Object.


Data Primitive adalah tipe data dasar yang biasanya telah tersedia oleh suatu bahasa pemrograman, sedangkan


Object adalah tipe data yang memiliki banyak value didalamnya.

Number

Tipe data yang dapat menampung nilai dalam bentuk float atupun integer.

String

Tipe data yang menampung nilai teks atuapun tulisan.

Boolean

Tipe data yang menampung nilai True atau False.

Undefined

Tipe data yang belum didefinisikan nilainya.

Null

Tipe data yang sifatnya kosong.
