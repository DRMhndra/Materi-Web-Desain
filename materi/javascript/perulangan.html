Setiap bahasa pemrograman memiliki konsep perulangan untuk melakukan intruksi tertentu sampai kondisi terpenuhi, sehingga tidak perlu menuliskan ulang code.


Terdapat 3 jenis perulangan dalam javascript

  • For
  • While
  • Do-while

For

For merupakan konsep perulangan dengan kondisi terbatas atau telah diketahui jumlah perulangan sebelumnya.

for(< awal >; < banyaknya perulangan >; < increment/decrement >){
 // statment yang akan dijalankan
}

While

Konsep pemrograman yang dimana jumlah perulangan belum diketahui sebelumnya.

while(< kondisi >){
 // statment yang akan dijalankan
}

Do-while

Konsep pemrograman yang sedikit berbeda dengan while dimana pengecekan kondisi diawal sedangkan do-while pengecekan kondisi di akhir sehingga perintah akan dijalankan minimal sekali baik itu salah ataupun benar.

do{
 // statment yang akan dijalankan
}while(< kondisi >);
