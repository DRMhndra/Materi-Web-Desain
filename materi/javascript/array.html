Array pada umumnya adalah kumpulan tipe data yang sama, namun untuk javascript dapat berupa campuran tipe data bisa string, number, dan object.


Array di dalam javascript termasuk dalam tipe data object, seperti array pada umumnya terdiri dari index dan value.

Membuat Array

let mahasiswa = new Array('sandi', 'dani', 'ahmad'); // cara lama
let mahasiswa = ['sandi', 'dani', 'ahmad']; // cara lebih singkat

Mengakses data array

Sama pada array umumnya untuk mengakses data menggunakan index dari isi array tersebut, dan index selalu dimulai dari 0.

let mahasiswa = ['sandi', 'dani', 'ahmad'];
console.log(mahasiswa[1]) // dani

Merubah value array

let mahasiswa = ['sandi', 'dani', 'ahmad'];
mahasiswa[1] = 'sulaiman';
console.log(mahasiswa[1]) // sulaiman

Manipulasi value pada array

Dikarenakan array adalah tipe data object dalam javascript sehingga terdapat method atau fungsi bawaan yang telah tersedia di object array yang dapat digunakan untuk manipulasi isi array tersebut, diantaranya:

forEach

forEach digunakan untuk melakukan perulangan pada isi dari suatu array.

array.forEach(function(< data >){
 //nilai yang akan diulang
});

 // data yang akan difilter
});

 // data yang akan dimanipulasi
});
