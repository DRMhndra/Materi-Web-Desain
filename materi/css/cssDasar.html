CSS atau singkatan dari Cascading Style Sheets digunakan untuk membuat tampilan suatu website lebih menarik dengan menambahkan desain.

Terminologi

Css bekerja dengan menseleksi suatu element html dengan selector lalu setelah itu akan dilakukan perubahan terhadap format, tulisan, tampilan dan lain sebagainya sesuai dengan keinginan.


Sebuah website jika tanpa diberikan style anggap saja seperti model kerangka tulang yang biasa terdapat di lab Biologi, namun dengan menambahkan css kita dapat memberikan tampilan supaya terkesan menarik.
