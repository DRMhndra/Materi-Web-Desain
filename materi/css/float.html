Float

Float adalah properti css yang akan membuat sebuah element berada di kiri ataupun kanan dari sebuah container.

Float biasa digunakan untuk membuat layouting sebuah website,karena mudah dipahami karena hanya memberi nilai left dan right pada sebuah element.

Permasalahan Pada Float

masalah pada float biasa terjadi karena element container(pembungkus) akan menggangap element di dalamnya yang menerapkan float dianggap tidak ada,terdapat 3 cara untuk mengatasi permasalahan ini.

Overflow

Dalam menerapkan metode overflow dapat menambahkan property overflow : auto di container(pembungkus) element yang di telah menerapkan float

Clear

Untuk menerapkan metode clear dengan cara menambahkan div setelah element yang menerapkan float dengan attribute class clear

Micro clearfix

Micro clearfix metode yang ditemukan oleh Nicolas Gallagher untuk memperbaiki masalah dalam menerapkan float,dengan metode ini kita hanya cukup memberi attribute class cf di container(pembungkus).
