Flexbox atau flexible box berguna untuk menyusun sebuah layout yang dapat mengontrol ukuran, tata letak, jarak dari sebuah element berdasarkan pembungkus(container) yang menampungnya.

Terminology Flexbox

Di dalam pembuatan layout menggunakan flexbox biasanya dikelompokan menggunakan pembungkus(container) yang biasa disebut flex-container, dan untuk element-element di dalamnya disebut dengan flex-items.

Penjelasan

  • main-axis = sumbu utama dari container arahnya berdasarkan properti direction
  • main-start | main-end = penempatan item pada container yang dapat diletakan antara sisi start dan juga end
  • cros-axis = sumbu silang dari container arahnya juga berdasarkan properti direction yang digunakan
  • cros-start | cros-end = penempatan item pada container

Arah dari sebuah flexbox tergantung dari bagaimana kita mendefinisikannya dimana terdapat 2 arah yaitu row dan column.

Ketika arah flexbox row maka main-axis adalah horizontal dan untuk crox-axis adalah vertical, sebaliknya ketika arah bernilai column.


Code untuk penjelasan selanjutnya


 

 

 


.container{
 display: flex;
}
.kotak {
 width: 100px;
 height: 100px;
}

Flexbox Property

flex-direction

flex-direction digunakan untuk menentukan arah dari element yang ada di dalam pembungkus(container).

.container{
 display: flex;
 flex-direction: row | row-reverse | column | column-reverse;
}
  • row kearah samping
  • row-reverse kearah samping namun item belakang menjadi didepan
  • column kearah bawah (tumpuk)
  • column-reverse kekearah bawah namun item bawah menjadi atas

flex-wrap

flex-wrap digunakan membuat item didalamnya apakah harus dibuat ke baris baru jika sudah memenuhi lebar dari containernya

.container{
 display: flex;
 flex-wrap: no-wrap | wrap | wrap-reverse;
}
  • no-wrap,tidak akan dibuat didalamnya ke baris baru (default)
  • wrap,dibuat ke baris baru
  • wrap-reverse dibuat baris baru namun terbalik

Terdapat syntax shorth-hand untuk kedua property diatas yaitu flex-flow

.container{
 flex-flow: flex-direction flex-wrap;
}

justify-content

justify-content digunakan untuk mengatur item-flexbox secara main-axis (horizontal/vertical) sesuai direction

.container{
 justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
}
  • flex-start,mengatur elemen ke arah samping kiri dari containernya
  • flex-end,mengatur elemen ke arah samping kanan dari containernnya
  • center,mengatur elemen ke tengah
  • space-between,mengatur elemen menjadi saling sama rata
  • space-around,mengatur elemen menjadi saling sama rata untuk main-axis namun memberikan ruang kosong yang sama diantara item
  • space-evenly,mengatur elemen menjadi saling sama rata untuk main-axis namun memberikan ruang kosong yang sama diantara item dan ruang tepi

align-items

align-items mengatur item-flexbox secara cros-axis (horizontal/vertical) sesuai dengan direction

.container{
 align-items: flex-start | flex-end | stretch | center | baseline;
}
  • flex-start,mengatur elemen ke arah atas dari containernya
  • flex-end,mengatur elemen ke arah bawah dari containernnya
  • stretch,mengatur elemen sama tinggi sesuai dengan containernya
  • center,mengatur elemen kearah tengah
  • baseline,mengatur elemen sesuai dengan text yang ada di item-flex

align-content

align-content digunakan untuk mengatur elemen jika lebih dari containernya berdasarkan cross-axis biasanya digunakan bersama dengan flex-wrap untuk property nya sama seperti justify-content dan align-items

.container{
 align-content: flex-start | flex-end | center | space-between | space-around | space-evenly | stretch | center | baseline;
}

flex-grow

flex-grow digunakan untuk menambahkan ukuran item jika memungkinkan pada container tersedia space kosong.

.container{
 flex-grow: < ukuran grow >;
}

flex-shrink

flex-shrink digunakan untuk menyusutkan sebuah item terhadap sisa ukuran container/browser.

.container{
 flex-shrink: < ukuran shrink >;
}

flex-basis

Flex-basis digunakan untuk mengatur ukuran awal dari sebuah item.

.container{
 flex-basis: < ukuran awal >;
}

order

Order digunakan untuk menentukan urutan dari flex-item tanpa harus merubah element html secara langsung.

.kotak-hijau{
 order : < urutan item >;
}
