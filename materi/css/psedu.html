Pseudo-class termasuk selector dalam css yang digunakan untuk mengatur property yang biasanya tidak "terlihat"

.link-text:hover { color: orange; }

Setiap element memiliki pseudo-class tersendiri namun bersifat opsional dan customize sesuai dengan Web Developer dalam menyusun layout apakah ingin menggunakna pseudo-class.

:hover

pseudo-class :hover digunakan jika ingin merubah sesuatu ketika cursor berada di atas element.

.link-text:hover {
 cursor: pointer;
 color: orange;
 text-decoration: underline;
}

:visited

pseudo-class yang satu ini digunakan biasanya terhadap tag link untuk memberi tahu bahwa link tersebut telah dikunjungi sebelumnya.

.link-text:visited {
 color: red;
 text-decoration: none;
}

:focus

Untuk pseudo-class yang satu ini digunakan bersamaan dengan element pengumpul informasi contohnya input, text-area dan juga button.

.form-control:focus {
 background-color: yellow;
 outline: none;
 border: 2px solid #cec8c8;
 border-radius: 5px;
}

Serta masih banyak lagi jika ingin melihat list pseudo-class yang dapat digunakan dapat berkunjung ke website ini
