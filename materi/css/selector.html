Selector

Selector adalah aturan untuk menentukan, memanipulasi format penulisan ataupun bentuk dari sebuah element html yang dipilih. Dalam menseleksi sebuah element dapat dibedakan menjadi 4 selector diantaranya

  1. Selector Universal
  2. Selector Tag
  3. Selector id atau class
  4. Selector berdasarkan attribute dari suatu element

Selector Universal

Tipe selector yang akan menerapakan semua aturan kedalam element html yang ada disuatu file.

<body>
 
Varian virus telah mencapai versi Tango

 
Virus kini kian meningkat...

</body>

lalu diberikan aturan universal css

* {
 color: lightpink;
 text-decoration: underline;
}

Varian virus telah mencapai versi Tango

Selector Tag

Selector tipe ini akan memberikan aturan ke dalam tag yang dipilih selama nilai (specificity) dari tag tersebut sama.

<body>
 
Paragraph ini akan berwarna biru

 
Virus kini kian meningkat...

 
Paragraph ini juga akan berwarna biru

</body>

diberikan aturan menggunan selector tag

p { color: blue; }

Paragraph ini akan berwarna biru

Paragraph ini berbeda sendiri

Paragraph ini juga akan berwarna biru

Lalu kenapa paragraph kedua tidak ikut berubah warna? itu di karenakan perbedaan berat bobot nilai atau (spesificity) dari suatu selector, spesificity akan dibahas materi yang berbeda.

Selector id dan class

Selector ini digunakan saat dalam suatu element terdapat salah attribute class ataupun id.

<body>
 

  

  

 

</body>

untuk attribute class di dalam selector menggunakan .< diikuti-dengan-nama class >

.kotak-kartu {
 border: 1px solid black;
 width: 150px; 
 height: 150px;
}
.kotak-header {
 height: 50px; 
 background-color: lightblue;
}

untuk attribute id di dalam selector menggunakan #< diikuti-dengan-nama id >

#kotak-body {
 border: 1px solid black;
 width: 150px; 
 height: 150px;
}
