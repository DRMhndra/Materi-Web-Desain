Display Element

Element html terdiri dari 3 display yaitu block,inline serta tambahan inline-block

Block

setiap element yang memiliki display block akan membuat element selanjutnya berada di bawah element sebelumnya,element yang memiliki display block diantaranya:

  • div
  • h1-h6
  • p
  • form
  • header
  • footer

Inline

Element yang memiliki display inline akan membuat element tersebut sejajar dengan elemen yang lain,element yang memiliki display inline antara lain:

  • a
  • img
  • span
  • code
  • em
  • dll

Contoh display inline

link ke pertemuan 1 link ke pertemuan 2

Inline-block

Element inline-block diartikan suatu element memiliki display campuran antara block dan inline.


Sifat inline-block adalah menyusun kesamping seperti inline tetapi kita dapat mengatur tingi dan lebar seperti block.
