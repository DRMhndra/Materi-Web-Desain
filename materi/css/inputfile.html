Penulisan CSS

Didalam menuliskan syntax CSS terdapat 2 hal yang di perlukan,yaitu


  1. Element yang akan menerapkan style yang biasa di sebut Selector
  2. Intruksi style yang akan di terapkan dalam element tersebut

Intruksi style css dapat menerapkan lebih dari 1 dengan memisahkan antara style dengan tanda ;

h1 { color : black; }
/*contoh style lebih dari 1*/
p {
 color: black;
 font-size: 20px;
}

Menambahkan file CSS kedalam HTML

Terdapat 3 cara untuk memasukan/menuliskan file csss,yaitu :


  1. Inline
  2. Internal
  3. Eksternal

Inline

Memasukan syntax css secara langsung ke dalam element html disebut dengan inline, untuk memasukannya menggunakan attribute style

Hello World

Hello World

Internal

memasukan syntax didalam file html tepatnya di antara tag head dengan menggunakan tag style disebut dengan Internal Css

<!DOCTYPE html>
<html>
<head>
 <title>Internal Css</title>
 <style>
  .box {
   width: 100px;
   height: 100px;
   background-color: grey;
  }
 </style>
</head>
<body>
 <div class="box"></div>
</body>
</html>

Eksternal

Memasukan syntax css yang berbeda folder ke dalam file html dengan menggunakan tag link yang di tulisakan diantara tag head

<!DOCTYPE html>
<html>
<head>
 <title>Eksternal Html</title>
 <link rel="stylesheet" href="folder-tujuan.css">
</head>
<body>
 <div class="kotak"></div>
</body>
</html>
