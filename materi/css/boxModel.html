Box Model

Setiap element html diibaratkan sebuah kotak layaknya kotak pada umumnya pasti terdapat isi,tinggi,lebar, hingga lapisan luar di Html kita menyebutnya padding,border,dan margin.

Secara default kotak akan mengikuti ukuran dari sebuah content di dalamnya, namun dapat diatur menggunakan css.

Content

Content area adalah tempat menempatkan content dari sebuah element yang dapat berupa text, image, table dan yang lainnya.

Padding

Padding area adalah bagian yang membatasi bagian antara border dari suatu element dengan content, ketebalan padding dapat ditentukan satu persatu ataupun menggunakan shorthand padding properties.

.box {
 padding-top: 10px;
 padding-right: 15px;
 padding-bottom: 20px;
 padding-left: 25px;
 
 padding: 10px 15px 20px 25px;
}

Border

Border area adalah garis yang mengelilingi content serta padding, ketebalan dan bentuk dari border itu sendiri pun dapat diatur satu persatu ataupun menggunakan shorthand border properties.

.box {
 border-top: 10px;
 border-right: 15px;
 border-bottom: 20px;
 border-left: 25px;
 
 border: 1px;
}

Margin

Margin area mengatur jarak antar element yang ada di dalam html ukuran margin dapat diatur satu persatu ataupun menggunakan shorthand margin properties, untuk margin dan padding sifatnya berbentuk transparan.

.box {
 margin-top: 10px;
 margin-right: 15px;
 margin-bottom: 20px;
 margin-left: 25px;
 
 margin: 10px 15px 20px 25px;
}
