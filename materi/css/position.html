Position

Position adalah property yang digunakan untuk mengatur posisi elemen didalam layout html, adapun property dapat dikategorikan seperti ini:

Property yang berifat Static
static
Property yang bersifat non-static
relative
absolute
fixed
sticky

Perbedaan Static dan Non-Static

Static

Secara default setiap element bersifat static ketika dibuat, yang mana akan mengikuti flow layout yaitu menumpuk kebawah ataupun memanjang kesamping.

Non-Static

Untuk element yang diatur non-static perilaku akan sedikit berbeda dikarenakan tidak lagi terikat oleh flow layout suatu dokument dapat diatur sedemikian rupa, ketika menerapkan non-static dapat menerapkan property tambahan top,left,right,dan bottom.

Property Position

Static

Seperti yang dijelaskan sebelumnya property: static akan mengikuti flow layout yaitu menumpuk kebawah ataupun memanjang kesamping.

Setiap element yang memiliki posisi static,maka setiap perubahan akan mempengaruhi element lainnya.

Relative

Element yang diberikan posisikan relative diibaratkan keluar dari flow normal.

.kotak-merah{
 position: relative;
}

Absolute

Sama seperti position: relative keluar dari flownya namun element yang menerapkan absolute tempat awal element akan dianggap tidak ada dan akan relative terhadap element terdekatnya.

.kotak-merah{
 position: absolute;
}

Fixed

posisi fixed akan dianggap keluar dan akan relatif terhadap jendela browser sehingga akan selalu melihat element tersebut di layar jendela browser.

.kotak-merah{
 position: fixed;
}

Sticky

Posisi Sticky adalah posisi gabungan antara posisi relative dan fixed,dimana posisi akan tetap berada di tempat awalnya namun akan menempel terhadap browser ketika sampai di element tersebut.
